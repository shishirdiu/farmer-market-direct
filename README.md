# KrishiBazar - Farmer to Market Direct Selling System

## About
A web-based platform connecting farmers directly to buyers, 
eliminating middlemen, with manual bKash payment verification, 
group buying, and demand forecasting features.

## Features
- Farmer/Buyer/Admin authentication
- Product CRUD (Farmer)
- Search/Filter products (JS DOM)
- Order placement with bKash payment verification
- Group Buying system
- Demand Board (demand forecasting)
- Admin panel

## Tech Stack
- HTML, CSS, JavaScript (DOM, OOP)
- PHP (OOP), MySQL

## Setup Instructions
1. Clone this repository
2. Import database/schema.sql into MySQL
3. Update config/database.php with your DB credentials
4. Run on XAMPP/WAMP local server
